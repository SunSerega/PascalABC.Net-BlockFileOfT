Описание:

080:///Инициализирует переменную файла, привязывая её к файлу fname
081:public constructor(fname:string) :=
082:Assign(fname);

Параметры:

fname: string

Задаёт имя файла, к которому будет привязан созданный экзепляр.

Примеры:

uses BlockFileOfT;

begin
var f: BlockFileOf<byte>;
f := new BlockFileOf<byte>(temp.bin);
end.

uses BlockFileOfT;

begin
var f := new BlockFileOf<byte>(temp.bin);
end.

uses BlockFileOfT;

begin
var f := BlockFileOf&<byte>.Create(temp.bin);
end.

Примечания:
Зачем нужны конструкторы

Привязку к файлу можно будет в любой момент изменить.
Но если вызывать этот конструктор, вместо пустого+Assign - надо меньше кода.
