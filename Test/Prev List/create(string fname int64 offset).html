Описание:

083:///Инициализирует переменную файла, привязывая её к файлу fname
084:///А так же устанавливая значение смещени от начала в байтах на offset
085:public constructor(fname:string; offset:int64) :=
086:Assign(fname, offset);

Параметры:

fname: string

Задаёт имя файла, к которому будет привязан созданный экзепляр.

offset: int64

Задаёт cмещение в байтах от начала файла до начала элементов

Примеры:

uses BlockFileOfT;

begin
var f: BlockFileOf<byte>;
f := new BlockFileOf<byte>(temp.bin, 5);
end.

uses BlockFileOfT;

begin
var f := new BlockFileOf<byte>(temp.bin, 5);
end.

uses BlockFileOfT;

begin
var f := BlockFileOf&<byte>.Create(temp.bin, 5);
end.

Примечания:
Зачем нужны конструкторы
Зачем нужен offset

Привязку к файлу и смещение можно будет в любой момент изменить.
Но если вызывать этот конструктор, вместо пустого+Assign+Offset - надо меньше кода.
