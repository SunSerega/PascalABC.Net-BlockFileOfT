Описание:

114: ///Достигнут ли конец файла
115: public property EOF:boolean read ByteSize-PosByte < sz;

Примеры:

uses BlockFileOfT;

begin
var f := new BlockFileOf<byte>('temp.bin');
f.Rewrite;
f.Size := 5;

f.Pos := 1;
writeln(f.EOF);//False

f.Pos := 5;
writeln(f.EOF);//True
end.

Примечания:
Это свойство указывает, можно ли прочитать ещё 1 элемент из файла, не получив ошибку
